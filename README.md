# AssetFlow

Enterprise Asset & Resource Management System built for **Odoo Hackathon 2026**.

## Overview

AssetFlow is designed to help organizations efficiently manage the complete lifecycle of assets—from registration and allocation to maintenance, audits, reporting, and notifications.

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

### Backend
- FastAPI
- PostgreSQL
- Raw SQL
- JWT Authentication
